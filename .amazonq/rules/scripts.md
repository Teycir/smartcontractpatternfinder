# Script Rules

**CRITICAL**: Maintain organized script structure.

## ✅ REQUIRED: Script Placement

- All shell scripts (`.sh`), Python scripts (`.py`), and other utility scripts MUST be saved in the `scripts/` folder.
- Do not clutter the root directory with utility scripts.

## 📋 Checklist

- [ ] New `.sh` files are in `scripts/`
- [ ] New `.py` files are in `scripts/`
