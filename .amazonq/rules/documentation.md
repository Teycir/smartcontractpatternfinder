# Documentation Rules

**CRITICAL**: Maintain organized documentation structure.

## ✅ REQUIRED: File Placement

- All `.md` files MUST be saved in the `docs/` folder.
- **EXCEPTIONS**: `README.md` and `CHANGELOG.md` MUST remain in the project root directory.

## 📋 Checklist

- [ ] New documentation files are in `docs/`
- [ ] Root directory only contains `README.md` and `CHANGELOG.md`
