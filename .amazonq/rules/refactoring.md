# Refactoring Rules

**CRITICAL**: Code must be continuously refactored to maintain quality.

## Immediate Refactoring Triggers
1. Function Length > 50 Lines
2. File Length > 300 Lines
3. Duplicate Code (DRY Violation)
4. Magic Numbers/Strings
5. Complex Conditionals (> 2 levels)
6. Side Effects in Pure Functions

## Best Practices
- Small steps
- Test first
- One thing at a time
- Commit often
- Measure impact
